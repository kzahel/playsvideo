## [0.0.1] - 2026-03-07

### Added
- Initial Chrome extension build (MV3)
- Popup window player (no address bar, native feel)
- File input and drag-and-drop support
- File handlers for Chrome OS (mp4, mkv, webm, mov, avi, ts)
- Dark theme UI
